indexOf(item) : item이 배열에 몇번째 위치에 있는지 검사한다
lastIndexOf(itme) : item이 배열에 몇번째 위치에 있는지 뒤에서 부터 검사
indexOf(item, start) : start위치부터 시작해서 item이 배열에 몇번째 위치에 있는지 검사한다

push(a,b,c...) : 배열의 마지막에 추가한다.
a = pop() : 배열의 마지막요소를 제거하고 리턴

reverse() : 배열의 순서를 뒤집기
sort() : 배열의 요소를 사전순으로 정렬

slice(start, end) : start ~ end 사이의 요소를 자른다
splice(index, n, a, b, c....) : index위치에서 n개를 지우고 a,b,c를 삽입

arr.concat(aa) : arr배열과 aa배열을 하나로 합쳐서 arr2배열을 생성
join() : 하나의 배열을 문자열로 변환
